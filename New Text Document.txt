PassengerId → Unique ID for each passenger (just a number, not a real feature).

Survived → Target variable: 0 = Did not survive, 1 = Survived.

Pclass → Passenger class: 1 = 1st class, 2 = 2nd class, 3 = 3rd class.

Name → Passenger’s name (includes title like Mr, Mrs, Miss, etc.).

Sex → Passenger’s gender (male, female).

Age → Passenger’s age in years (can contain missing values).

SibSp → Number of siblings/spouses the passenger had aboard.

Parch → Number of parents/children the passenger had aboard.

Ticket → Ticket number (not very useful, but sometimes used).

Fare → Ticket fare (the amount the passenger paid).

Cabin → Cabin number (many missing values).

Embarked → Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton).
